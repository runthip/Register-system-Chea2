# ระบบรับสมัครนักเรียนออนไลน์ 2569
### โรงเรียนเจียรวนนท์อุทิศ 2

ระบบนี้ใช้ HTML/Tailwind CSS สำหรับหน้าเว็บ (Frontend) และ Google Apps Script สำหรับระบบหลังบ้าน (Backend)

## วิธีการใช้งาน
1. นำไฟล์ `index.html` อัปโหลดขึ้น GitHub Repository
2. ตั้งค่า GitHub Pages เพื่อเปิดใช้งานเว็บไซต์
3. นำโค้ดใน `google-apps-script.gs` ไปวางใน Google Apps Script
4. เปลี่ยน `URL_WEB_APP` และ `FOLDER_ID` ในโค้ดให้เป็นของคุณเอง
